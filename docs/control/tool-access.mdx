# Tool Access Control

Define which agents can use which tools.

## Usage

```python
from splinter.control import ToolAccessController

controller = ToolAccessController()

# Allow specific tools for each agent
controller.set_allowed_tools("researcher", ["web_search", "read_file"])
controller.set_allowed_tools("writer", ["write_file"])

# Check access before tool call
controller.check_access("researcher", "web_search")  # OK
controller.check_access("writer", "web_search")       # Raises error
```

## Deny Lists

```python
# Block specific tools
controller.set_denied_tools("junior_agent", ["delete_file", "execute_code"])
```

## Tool Registry

```python
from splinter.control import ToolRegistry

registry = ToolRegistry()

# Register tools with metadata
registry.register(
    name="web_search",
    description="Search the web",
    requires_approval=False,
)

registry.register(
    name="delete_file",
    description="Delete a file",
    requires_approval=True,  # Dangerous operation
)
```

## Error Handling

```python
from splinter.exceptions import ToolAccessDeniedError

try:
    controller.check_access("writer", "web_search")
except ToolAccessDeniedError as e:
    print(f"Agent '{e.agent_id}' cannot use tool '{e.tool_name}'")
```

## With Gateway

```python
from splinter import Gateway
from splinter.control import ToolAccessController

controller = ToolAccessController()
controller.set_allowed_tools("researcher", ["web_search"])

gateway = Gateway(tool_access=controller)
# Gateway automatically checks tool access
```
