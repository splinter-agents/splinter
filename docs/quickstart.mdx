## Installation

```bash
pip install -e .
pip install openai  # or: anthropic, google-generativeai
```

## Simple API (Recommended)

```python
from splinter import Splinter

# One line setup
s = Splinter(openai_key="sk-...")

# Run an agent
result = await s.run("helper", "What is 2+2?")
print(result)

# Check cost
print(f"Cost: ${s.cost:.4f}")
```

## Multi-Agent Pipeline

```python
from splinter import Splinter

s = Splinter(openai_key="sk-...")

# Define agents
s.add_agent("researcher", "Research topics. Output JSON: {findings: [...]}")
s.add_agent("writer", "Write articles. Output JSON: {article: '...'}")

# Run pipeline
result = await s.pipeline(["researcher", "writer"], input={"topic": "AI"})

print(result.outputs["writer"])
print(f"Cost: ${s.cost:.4f}")
```

## Sync Usage

```python
from splinter import Splinter, run_sync

s = Splinter(openai_key="sk-...")
result = run_sync(s.run("helper", "What is 2+2?"))
```

## With Limits

```python
s = Splinter(
    openai_key="sk-...",
    max_budget=1.0,   # Stop at $1
    max_steps=20,     # Stop at 20 calls
)
```

## Any Provider

```python
# OpenAI
s = Splinter(openai_key="sk-...")

# Anthropic (Claude)
s = Splinter(anthropic_key="sk-...")

# Gemini
s = Splinter(gemini_key="...")

# Grok
s = Splinter(grok_key="xai-...")

# Or use env vars
s = Splinter()  # Reads from OPENAI_API_KEY, ANTHROPIC_API_KEY, etc.
```

## What Just Happened?

Behind the scenes, Splinter's Gateway:

1. **Control checks** - Budget OK? Steps OK? Loop detected?
2. **Routes to LLM** - OpenAI, Anthropic, Gemini, or Grok
3. **Coordination** - Updates state, creates checkpoint

```
Agent → GATEWAY → Control ✓ → LLM → Coordination → Response
```
