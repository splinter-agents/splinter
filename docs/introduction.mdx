# Splinter

**Stop agents from going rogue. Enforce limits. Coordinate state. Resume from failures.**

```python
from splinter import Splinter

s = Splinter(openai_key="sk-...")
result = await s.run("researcher", "Find info about AI")
```

That's it. All safety controls are built-in.

## The Problem

Agents without guardrails will:
- Spend unlimited money on API calls
- Loop forever on the same task
- Overwrite each other's work
- Crash with no way to resume

## The Solution

Splinter's **Gateway** intercepts every LLM call:

```
Agent → GATEWAY → Control ✓ → LLM → Coordination → Response
```
